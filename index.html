🤝 In-Person Interview / Meeting
Phone table pe rakh, volume max. Record karo → English plays from speaker → interviewer directly sunta hai. No setup needed.
💻 Google Meet / Zoom on Laptop
1. Download free VB-Audio Virtual Cable — vb-audio.com/Cable
2. Meet → Settings → Mic → select CABLE Output (VB-Audio)
3. Windows Sound → Playback → set CABLE Input as default
4. Open BolO on same laptop. Record → English TTS plays through virtual cable → Meet sends it as your mic voice. Other party hears English.
📱 Meet / Zoom on Phone
2-device trick (best): Phone 1 = Meet call. Phone 2 = BolO open. Record on Phone 2 → English plays on speaker → Phone 1 mic picks it up → other party hears English.

Single phone: Record → Copy button → paste English text in Meet chat.
📞 Regular Phone Call
Speaker mode ON on call phone. Open BolO on same or second phone. Record → English plays from BolO speaker → call mic picks it up.
💡 Pro Tips
• Wear earphones while on call — prevents echo
• Speak clearly, complete sentence, then release
• Use Tap mode for longer sentences
• Use Hold mode for quick short phrases
